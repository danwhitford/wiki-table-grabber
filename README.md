
```
Usage of wiki-table-grabber:
  -n int
    	Index of table to print. Starts at 0. -1 prints all tables (default -1)
  -o string
    	If set will output to file(s). Will be in the form of <file>_<n>.csv if mulitple tables
  -s string
    	The CSS selector to identify tables. (default "table.wikitable")
```
